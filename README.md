## What's here

This repository is the open-source mirror of **[docs.ofmapi.com](https://docs.ofmapi.com)** — the
official documentation for the OFMAPI OnlyFans API. We mirror it here so the
community can:

- Suggest improvements via PR
- Spot-fix typos without opening a support ticket
- Use the MDX content as starter material for related documentation
- Track every change in git history

## What's documented

- 🚀 **Quickstart** — get from sign-up to your first webhook in 5 minutes
- 📖 **API reference** — every endpoint, every parameter, every response
  shape, with code examples in 6 languages
- 🔔 **Webhook event catalog** — every event your app can receive, with
  full example payloads
- 🛡️ **Authentication** — API keys, scopes, IP allowlists, idempotency
- 🤖 **MCP integration** — Claude / ChatGPT / Cursor / Manus setup guides
- 🧩 **No-code integrations** — n8n, Make, Zapier with example workflows
- 📦 **SDK guides** — Python, Node / TypeScript, Go
- 💼 **Use-case recipes** — chatbot, CRM, mass messaging, AI agent,
  earnings dashboard, traffic attribution
- 🔐 **Security & trust** — encryption, account safety, SOC 2 status,
  vulnerability reporting
